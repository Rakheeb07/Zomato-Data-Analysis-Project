# Zomato Data Analysis Using Python

## Project Description
This project performs Exploratory Data Analysis (EDA) on a Zomato-like restaurant dataset to understand customer behavior, restaurant performance, ratings, and ordering patterns. The analysis focuses on answering business-oriented questions using data and generating meaningful insights.

This is a pure Data Analysis project implemented using Python.

---

## Objectives
- Analyze customer preferences across different restaurant types  
- Understand voting and rating patterns  
- Study online vs offline ordering behavior  
- Analyze spending trends of customers  
- Support data-driven business decisions  

---

## Questions Answered
The following business questions are addressed in this project:

1. What type of restaurant do the majority of customers order from?  
2. How many votes has each type of restaurant received from customers?  
3. What ratings do most restaurants receive?  
4. What is the average spending per order for couples who mostly order online?  
5. Which ordering mode (online or offline) has received the maximum ratings?  
6. Which type of restaurant receives more offline orders so that suitable offers can be provided?  

---

## Approach
The project follows a structured data analysis workflow:

### Data Cleaning
- Removed duplicate records  
- Handled missing values  
- Corrected data types  
- Cleaned categorical columns  

### Exploratory Data Analysis (EDA)
- Distribution analysis of restaurant types  
- Voting and rating comparison  
- Spending behavior analysis  
- Online vs offline order comparison  
- Data visualizations for better insights  

### Insight Generation
- Identified trends and patterns from data  
- Converted analytical results into business insights  

---

## Key Insights
- Certain restaurant types receive more customer orders  
- Voting and rating patterns vary across restaurant categories  
- Most restaurants fall within a common rating range  
- Couples ordering online show a consistent average spending trend  
- Online ordering mode generally receives higher ratings  
- Some restaurant types rely more on offline orders and can be targeted with specific offers
